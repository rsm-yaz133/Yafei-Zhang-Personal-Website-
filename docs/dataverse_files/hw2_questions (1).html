Introduction

Angrist and Lavy (1999) exploit a natural experiment arising from Maimonides’ Rule — an administrative rule in Israel that caps class sizes at 40 students. When enrollment crosses a multiple of 40, an additional class is created, producing a sharp, discontinuous drop in predicted class size. The authors use this rule as an instrumental variable to estimate the causal effect of class size on student achievement (reading and math scores) for 4th and 5th graders in Israeli public schools.

The key identification strategy is that Maimonides’ Rule creates exogenous variation in actual class size: schools with enrollment just above a cutoff (e.g., 41, 81, 121) are predicted to have smaller classes than schools just below. This allows for a regression-discontinuity / instrumental-variables approach that overcomes the endogeneity problem inherent in OLS estimates of class size effects.

This document replicates Tables II–V from the published paper using R.

Data Loading and Cleaning

The data consist of class-level observations for 4th and 5th graders in Israeli public schools. The cleaning steps follow the original Stata replication code exactly.

Code 
# Function to load and clean data (mirrors the Stata code)
clean_data <- function(filepath) {
  df <- read_dta(filepath, encoding = "latin1")

  # Fix scores > 100 (same as Stata: replace avg = avg - 100 if avg > 100)
  df$avgverb[df$avgverb > 100 & !is.na(df$avgverb)] <-
    df$avgverb[df$avgverb > 100 & !is.na(df$avgverb)] - 100
  df$avgmath[df$avgmath > 100 & !is.na(df$avgmath)] <-
    df$avgmath[df$avgmath > 100 & !is.na(df$avgmath)] - 100

  # Maimonides' Rule: predicted class size
  # func1 uses c_size (enrollment), func2 uses cohsize
  df$func1 <- df$c_size / (floor((df$c_size - 1) / 40) + 1)
  df$func2 <- df$cohsize / (floor(df$cohsize / 40) + 1)

  # Set scores to NA if test sample size is 0
  df$avgverb[df$verbsize == 0] <- NA
  df$passverb[df$verbsize == 0] <- NA
  df$avgmath[df$mathsize == 0] <- NA
  df$passmath[df$mathsize == 0] <- NA

  # Sample restrictions (same as Stata)
  df <- df[df$classize > 1 & df$classize < 45 & df$c_size > 5, ]
  df <- df[df$c_leom == 1 & df$c_pik < 3, ]

  return(df)
}

run_table3 <- function(df, dvar, samp_label) {
  if (samp_label == "all") {
    d <- df
  } else {
    d <- df[df$disc == TRUE, ]
  }
  m1 <- feols(as.formula(paste(dvar, "~ func1 + tipuach")), data = d, cluster = ~schlcode)
  m2 <- feols(as.formula(paste(dvar, "~ func1 + tipuach + c_size")), data = d, cluster = ~schlcode)
  list(m1, m2)
}

Table IV: 2SLS Estimates – 5th Grade

Table IV presents the main IV results for 5th graders, instrumenting actual class size with predicted class size from Maimonides’ Rule (func1). Specifications progressively add enrollment controls, a quadratic in enrollment, and a piecewise-linear trend.

Code 
# Additional variables for Table IV/V
g5_t3$c_size2 <- (g5_t3$c_size^2) / 100

# Piecewise linear trend (as in Stata code)
g5_t3$trend <- NA
g5_t3$trend[g5_t3$c_size >= 0 & g5_t3$c_size <= 40]   <- g5_t3$c_size[g5_t3$c_size >= 0 & g5_t3$c_size <= 40]
g5_t3$trend[g5_t3$c_size >= 41 & g5_t3$c_size <= 80]   <- 20 + g5_t3$c_size[g5_t3$c_size >= 41 & g5_t3$c_size <= 80] / 2
g5_t3$trend[g5_t3$c_size >= 81 & g5_t3$c_size <= 120]  <- 100/3 + g5_t3$c_size[g5_t3$c_size >= 81 & g5_t3$c_size <= 120] / 3
g5_t3$trend[g5_t3$c_size >= 121 & g5_t3$c_size <= 160] <- 130/3 + g5_t3$c_size[g5_t3$c_size >= 121 & g5_t3$c_size <= 160] / 4

run_table4 <- function(df, dvar) {
  # Full sample
  m1 <- feols(as.formula(paste(dvar, "~ tipuach | classize ~ func1")),
              data = df, cluster = ~schlcode)
  m2 <- feols(as.formula(paste(dvar, "~ tipuach + c_size | classize ~ func1")),
              data = df, cluster = ~schlcode)
  m3 <- feols(as.formula(paste(dvar, "~ tipuach + c_size + c_size2 | classize ~ func1")),
              data = df, cluster = ~schlcode)
  m4 <- feols(as.formula(paste(dvar, "~ trend | classize ~ func1")),
              data = df, cluster = ~schlcode)

  # Discontinuity sample
  disc_df <- df[df$disc == TRUE, ]
  m5 <- feols(as.formula(paste(dvar, "~ tipuach | classize ~ func1")),
              data = disc_df, cluster = ~schlcode)
  m6 <- feols(as.formula(paste(dvar, "~ tipuach + c_size | classize ~ func1")),
              data = disc_df, cluster = ~schlcode)

  list(full1 = m1, full2 = m2, full3 = m3, full4 = m4, disc1 = m5, disc2 = m6)
}

Visualizing Maimonides’ Rule: The First Stage

To build intuition for the identification strategy, this plot shows how Maimonides’ Rule creates discontinuous jumps in predicted class size at multiples of 40. The sawtooth pattern in predicted class size provides the exogenous variation used for identification.

Code 
# Collapse to enrollment-level means
library(stats)
agg5 <- aggregate(cbind(classize, func1) ~ c_size, data = g5_t3, FUN = mean)

plot(agg5$c_size, agg5$classize,
     pch = 16, cex = 0.6, col = "steelblue",
     xlab = "Enrollment count", ylab = "Class size",
     main = "Actual vs. Predicted Class Size (5th Grade)",
     ylim = c(10, 45))
lines(agg5$c_size[order(agg5$c_size)], agg5$func1[order(agg5$c_size)],
      col = "firebrick", lwd = 2)
abline(v = c(40, 80, 120), lty = 2, col = "gray60")
legend("topright", legend = c("Actual class size (mean)", "Maimonides' Rule prediction"),
       pch = c(16, NA), lty = c(NA, 1), lwd = c(NA, 2),
       col = c("steelblue", "firebrick"), bty = "n")

This figure illustrates the key mechanism: at enrollment counts just above 40, 80, and 120, Maimonides’ Rule predicts a sharp drop in class size as the school must split into additional classes. The actual class sizes broadly follow this pattern but with considerable noise — exactly the kind of variation that makes a strong first stage for the IV strategy.
